# 3D Viewport

A simple 3D Viewer made with Three.js. Hover over the object button in the sidebar to highlight the correspondent object and click it to hide it.

## Made With
1. Three.js
2. Bootstrap 5 Alpha
3. Sass
4. Parcel
5. Node.js/Express

## Notes
1. No JS Framework used - except Three.js
2. Modular
3. ES6
